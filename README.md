# fakebook

## Info
Based on facebook the app has the following functionalities:
1. Create account
2. Make a post
3. Like a post
4. Comment on a post
5. Search for people
6. Send a friend request to people
7. Accept a friend request
8. Delete a friend
9. Edit your bio
10. View your friends' posts

## Stack
1. Node.js
2. React.js
3. Django - python
4. Postgresql via psycopg2
